# API REST para Gerenciamento de Usuários

## Tecnologias

- Node.js
- Express
- SQLite3

## Como rodar

1. Clone o repositório:
```bash
git clone https://github.com/Gabriel-Milani/API_REST_web3_CP3029514.git
cd api-usuarios

2. Instale as dependências:
npm install

3. Execute a aplicação:
node src/app.js

4. A API estará rodando em:
http://localhost:3000
